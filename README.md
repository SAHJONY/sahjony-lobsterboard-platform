# Sahjony Capital LLC - Lobsterboard UI

Real Estate Wholesale Platform with Lobsterboard Dashboard

## Live Deployment

The Lobsterboard UI is deployed and accessible at:

**🌐 LIVE URL:** https://sahjony.github.io/sahjony-lobsterboard-platform/

## Features

- **Lobsterboard Dashboard**: Enterprise-grade real estate wholesale dashboard
- **AI Agent Integration**: Acquisition, Disposition, and Cash Buyers agents
- **CRM Integration**: Complete customer relationship management
- **Bland.ai Call Automation**: Autonomous outbound/inbound calling with $55.16 balance
- **Portfolio Management**: Real-time property portfolio tracking
- **Multi-Agent Architecture**: Fortune 500-level agent orchestration

## Technology Stack

- **Frontend**: React/TypeScript with TanStack Start SSR
- **UI Components**: LobsterClaw-inspired enterprise dashboard
- **AI Integration**: Hermes multi-agent intelligence platform
- **Deployment**: GitHub Pages (static hosting)

## Access

To access the Lobsterboard UI:

1. Visit: https://sahjony.github.io/sahjony-lobsterboard-platform/
2. Navigate to `/real-estate` route for Lobsterboard dashboard
3. Skip onboarding by setting localStorage: `localStorage.setItem('hermes-onboarding-complete', 'true')`

## Repository Structure

- `src/components/` - React components including LobsterClawDashboard
- `src/routes/` - TanStack Start routes including real-estate.tsx
- `dist/client/` - Built static files for deployment
